<h2>Whiteboard Web App (Single HTML File)</h2>

<p>
A modern, lightweight whiteboard application built entirely using a single HTML file with embedded CSS and JavaScript. 
This tool provides a smooth and interactive drawing experience directly in the browser without relying on any external libraries or backend services.
</p>

<h3>Features</h3>
<ul>
  <li>Full-screen responsive drawing canvas</li>
  <li>Freehand drawing with mouse and touch support</li>
  <li>Pen and eraser tools for easy editing</li>
  <li>Custom color picker and adjustable brush size</li>
  <li>Undo and redo functionality for better control</li>
  <li>Clear canvas option</li>
  <li>Text tool with customizable size and color</li>
  <li><strong>Recent Text History:</strong> Previously used text appears in a top bar and can be reused with one click</li>
  <li>Download whiteboard as an image (PNG)</li>
  <li>Fully responsive and touch-friendly design</li>
</ul>

<h3>User Experience</h3>
<p>
The application is designed with a clean and modern interface, featuring smooth interactions, rounded elements, and an intuitive toolbar. 
The top bar enhances productivity by allowing quick reuse of recently entered text, making repeated annotations faster and easier.
</p>

<h3>Technology</h3>
<ul>
  <li>Pure HTML for structure</li>
  <li>Embedded CSS for styling</li>
  <li>Vanilla JavaScript for all functionalities</li>
  <li>Canvas API for drawing operations</li>
  <li>LocalStorage for saving recent text history</li>
</ul>

<h3>Usage</h3>
<p>
Simply open the HTML file in any modern web browser to start using the whiteboard instantly. 
No installation or setup is required.
</p>

<h3>Highlights</h3>
<ul>
  <li>No dependencies or external frameworks</li>
  <li>Runs completely offline</li>
  <li>Fast, lightweight, and easy to use</li>
  <li>All features packed into a single file</li>
</ul>
